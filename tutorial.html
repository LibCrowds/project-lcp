This page contains instructions on how to enter the information found on each of the four groups of cards. The four groups of cards are as follows:

  • 1824-1951
  • 1852-1899
  • 1900-1968
  • Correspondence Cards

Accents

A box containing a set of latin characters with accents will appear whenever you focus on a title or author field. Please click on the relevant characters to input text with the required accents. Alternatively, you can use the Character Map tool built into Windows and Mac OSX.

1824-1951

Title/s: Please write the article at the front in natural language, for example, 'The Dressmaker' rather than 'Dressmaker, The'. If there are two titles, please include both, separated by a semi-colon.

Cross reference cards

Some cards only feature two titles and no other information. In such cases just write the two titles in the title field, separated by a semi-colon.

Author/s: If the author is noted, please include it here. If two authors are noted, please include both, separated by a semi-colon.

Date: Please include the date, in the format shown on the card.

Reference Number: The reference number should be 5 digits and a folio reference (e.g. '45895 ff.941-983').

License Number and/or Year and Volume: There are no license numbers on these cards.

Other Information: Please transcribe any other information here.

1852-1899

Title/s: Please write the article at the front in natural language, for example, 'The Dressmaker' rather than 'Dressmaker, The'. If there are two titles, please include both, separated by a semi-colon.

Cross reference cards

Some cards only feature two titles and no other information. In such cases just write the two titles in the title field, separated by a semi-colon.

Author/s: If the author is noted, please include it here. If two authors are noted, please include both, separated by a semi-colon.

Date: Please include the date, in the format shown on the card.

Reference Number: The reference number should be 5 digits and a capital letter (e.g, '45895 B').

License Number and/or Year and Volume: If the license number is present, please include it here (e.g. 'License 56').

Other Information: Please transcribe any other information here.

1900-1968

Title/s: Please write the article at the front in natural language, for example, 'The Dressmaker' rather than 'Dressmaker, The'. If there are two titles, please include both, separated by a semi-colon.

Author/s: If the author is noted, please include it here. If two authors are noted, please include both, separated by a semi-colon.

Date: Please type the date in the following format: '6 Mar 1953'.

Reference Number: The reference number is 5 digits and a capital letter, eg '45895 B'. If no number is present, please leave this field blank.

License Number and/or Year and Volume: Transcribe the year and volume in this field, separated by a slash (e.g. '1942/17').

Other Information: Please transcribe any other information here.

Correspondence Cards

Title/s: Please write the article at the front in natural language, for example, 'The Dressmaker' rather than 'Dressmaker, The'. If there are two titles, please include both, separated by a semi-colon.

Author/s: If the author is noted, please include it here. If two authors are noted, please include both, separated by a semi-colon.

Date: Please type the date in the following format: '6 Mar 1953'.

Reference Number: The reference numbers are not included on these cards, please leave this field blank.

License Number and/or Year and Volume: Transcribe the year and/or license number here, separated by a slash (e.g. '1942/117') or if there is only a license number 'License number 2209'

Other Information: Please transcribe all other information here, including the notes about the play and any additional notes on the back of the card.
